# Restaurante
1. Fiz a tabela no Supabase criando a tabela de produtos, clientes e pedidos.
2. Criei a pasta do Backend, colocando o server.js.
3. Criei a pasta do frontend colocando o index.html e o style.css.
4. Instalei o npm init -y e o npm install express no backend.
5. No server.js coloquei as rotas e para iniciar o navegador.
6. Testei o servidor no Thunder rodando node server.js.
7. Instalei npm install cors dotenv @supabase/supabase-js para conectar o supabase.
8. Criei o arquivo .env e adicionei minhas chaves da tabela do supabase.
